MindAR Image Tracking Test

This app uses MindAR for image-based AR tracking.

Unlike barcode/fiducial markers, MindAR recognizes the full sticker image (QR code + barcode combined).

How it works:

  • Compile: In the editor, click "Compile AR Target" to analyze the sticker image and create a .mind tracking file
  • Print: Print the sticker and apply it to the panel
  • Scan: The technician scans the QR code, which opens the AR viewer
  • Track: MindAR recognizes the printed sticker and overlays annotations
